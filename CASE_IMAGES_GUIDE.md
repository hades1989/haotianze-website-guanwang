# 案例图片上传指南

## 使用方法

### 1. 上传图片

访问图片上传页面：`/admin/upload`

按照以下步骤上传图片：
1. 选择对应的案例
2. 点击上传区域或拖拽图片
3. 点击"上传图片"按钮
4. 上传成功后会显示图片路径

### 2. 更新案例页面

上传成功后，需要手动更新 `src/app/cases/page.tsx` 文件中的图片路径。

#### 示例：

假设你上传了案例ID为1的图片，返回的路径为 `/cases/case_1_1712345678900.jpg`

找到 `src/app/cases/page.tsx` 中对应的案例数据：

```typescript
{
  id: 1,
  title: '某国有银行省级分行安防系统升级',
  // ...
  image: 'https://images.unsplash.com/photo-1486406146926-c627a92ad1ab?w=800&h=600&fit=crop',
}
```

将 `image` 字段更新为新上传的路径：

```typescript
{
  id: 1,
  title: '某国有银行省级分行安防系统升级',
  // ...
  image: '/cases/case_1_1712345678900.jpg',
}
```

### 3. 图片要求

- **格式**：JPG、PNG、WEBP
- **大小**：最大 5MB
- **建议尺寸**：800x600 或 1200x800
- **建议文件命名**：case_{案例ID}_{时间戳}.{扩展名}

### 4. 图片存储位置

上传的图片保存在：`public/cases/` 目录

### 5. 注意事项

- 同一案例可以上传多张图片，建议使用有意义的文件名
- 上传后无需重启服务器，图片会自动生效
- 删除图片需要手动从 `public/cases/` 目录中删除
- 建议定期备份上传的图片

## 技术说明

### 上传 API

- **路径**：`/api/upload`
- **方法**：POST
- **参数**：
  - `file`：图片文件
  - `caseId`：案例ID

### 返回格式

```json
{
  "success": true,
  "imageUrl": "/cases/case_1_1712345678900.jpg",
  "filename": "case_1_1712345678900.jpg"
}
```
